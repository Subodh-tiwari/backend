Demo backend
-----------

Demo project for rabbitmq, redis, postgres, hapijs and jwttoken

Written in TypeScript

Prerequisites:
  * Rabbitmq
  * Redis
  * Postgres


Setup
-----

  **For development**

    npm install -g typescript typings

  **For running producers**

    npm install
    npm run start

  **For running consumers**

    npm install
    npm run start
